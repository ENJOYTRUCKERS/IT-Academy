# IT Academy プロジェクト 実装結果ウォークスルー

これまでに実装および完了した内容のウォークスルーです。

## 1. プロジェクト基盤の構築 (Phase 1)
- **Next.js 16 (App Router)** の環境を構築しました。
- **Tailwind CSS v4** を導入し、UIデザインガイドラインに沿ったブランドカラー（背景、文字、アクセント色）をセットアップしました。
- `@serwist/next` を使用し、インストール可能な **PWA (Progressive Web App)** 機能を設定しました。

## 2. 認証・ユーザー管理基盤の実装 (Phase 2)
`.env.local` に設定いただいた Supabase キーを用いて、安全な認証機能を構築しました。

### 年齢確認とログインフロー
- `/login` ページにアクセスすると、まず年齢確認が表示されます。
- **12歳以下の場合**:
  - GitHubログインはできず、「ひみつのじゅもん」を入力する画面になります。
  - バックエンドでは Supabase のメール＆パスワード認証を応用し、「`kid_<じゅもん>@truckers.jp`」のようなダミーアドレスとパスワードとして「ひみつのじゅもん」を利用する仕組みになっています。これにより、通常のユーザーと同じセキュアなインフラにデータを保存・復元できます。
- **13歳以上の場合**:
  - GitHub でのログインボタンが表示されます（Supabase OAuth連携）。

### 匿名ユーザー名の生成機能
- `src/utils/nameGenerator.ts` を作成し、「形容詞＋動物名＋数字（例: かわいいネコ4821）」で構成される完全匿名のユーザー名をランダムに生成するロジックを実装しました。

## 3. コア機能の実装 (Phase 3)

### 科目学習のホーム画面
- `src/app/page.tsx` を実装し、言語学習からドライビングアカデミーまで全7カテゴリの美しい科目一覧（グラスモーフィズムデザイン）を表示するようにしました。

### Web IDE (統合開発環境)
- `src/app/ide/` 以下に **Web IDE** を実装し、2つのモードをタブで切り替えられるようにしました。
  1. **コードモード**: `@monaco-editor/react` (VS Codeと同じエンジン) を使用し、本格的なコードエディタを搭載。
  2. **ブロックモード**: `blockly` (Google Blockly) を搭載し、視覚的にプログラミングを学べる機能を実装しました。

### 各種プレースホルダーの設置
- **質問掲示板 (`/board`)**:
  - 13歳以上18歳未満のユーザーに対して「閲覧のみ」となるように表示を出し分けるロジックを組み込みました。
- **ランキング (`/rankings`)**:
  - タイピングスコアや科目達成度のための表示枠組みを作成しました。

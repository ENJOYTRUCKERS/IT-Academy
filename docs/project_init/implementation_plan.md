# IT Academy Powered by ENJOYTRUCKERS プロジェクト初期化

「IT Academy Powered by ENJOYTRUCKERS」の新規Webアプリケーションプロジェクト立ち上げに向けたシステム方針および実装計画書です。

## 概要

- **サービス名**: IT Academy Powered by ENJOYTRUCKERS
- **URL**: https://academy.truckers.jp/
- **目的**: 小学生から大人まで、幅広くIT関連について学習するWEBアプリ
- **ターゲット**: 初心者〜上級者まで。小学生から大人まで。

## システムアーキテクチャ方針

### フロントエンド / バックエンド
- **フレームワーク**: Next.js (App Router) + TypeScript
  - 理由: PWA (Progressive Web App) 対応が容易であり、SEOやパフォーマンスに優れているため。また、Web IDEなど複雑なコンポーネントの組み込みに適している。
- **スタイリング**: Tailwind CSS
  - 理由: 提供されたUIデザインガイドライン（グラスモーフィズム、ホバーアニメーション、ダーク/ライトモード対応）を効率的に実装するため。

### データベース & 認証
- **BaaS**: Supabase または Firebase
  - 理由: GitHub連携ログイン、リアルタイムなランキング機能や掲示板機能を迅速かつセキュアに構築可能。
- **認証**: 
  - GitHub OAuth ログインのみ
  - 年齢確認のフローをサインアップ/初回ログイン時に組み込む。

### PWA (Progressive Web App) 対応
- `next-pwa` または `@serwist/next` などのライブラリを活用し、各端末（PC/スマホ）やブラウザからネイティブアプリのようにインストール可能にする。

### 匿名性・安全性に関する方針
- すべてのユーザーは匿名（例: ランダムな動物名＋数字 など）で利用。ユーザー名の手動設定不可。
- 13歳未満: 
  - GitHubログイン不可。
  - 進捗データはサーバーに保存しつつ、ローカルストレージに一意のキーを持たせてユーザーを識別。
  - **キャッシュ削除時等のデータ復元手段として、初回登録時に「ひみつのじゅもん（シークレットコード）」を発行し、これを入力することで別端末やキャッシュクリア後でも進捗を復元できるようにする。**
- 13歳以上18歳未満:
  - 掲示板は閲覧のみ（書き込み不可）。
- 18歳以上:
  - すべての機能を利用可能。

## 提案事項 (Feature Proposals)

- **タイピングゲームの組み込み**: タイピングスコアランキングを活性化させるため、IT用語やプログラミングのコードスニペットを入力する専用のタイピングゲーム機能の追加。
- **バッジ・アチーブメント機能**: 科目クリアごとに「基本情報マスター」「HTMLビギナー」のようなデジタルバッジを付与し、学習のモチベーションを向上させる。
- **チュートリアルモード**: 初回アクセス時に、マウスやキーボードの使い方、アプリのインストール方法をゲーム感覚で学べるオンボーディングの実装。
- **動物名+アバター生成**: 匿名ユーザー名だけでなく、可愛い動物のアイコンを自動割り当てすることで、子供でも親しみやすいUIにする。
